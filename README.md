
Tecnológias utilizadas:

Typescript;
ReactJS;
Styled-Components;
Eslint;
Prettier;

### Instruções:

`git clone https://github.com/brunoscardoso/-ReactJS-Compasso-GithubSearch-Test fronted`

`cd frontend`

`yarn install`

Depois que terminar a instalação é só rodar um dos comandos abaixo:

`yarn start`
